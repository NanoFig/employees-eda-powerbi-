##  Project Overview
Exploratory Data Analysis of an employees dataset to understand workforce composition,
salary distribution, departments, and potential insights for HR decision-making.

##  Objectives
- Explore employee demographics and structure
- Identify patterns in salaries and departments
- Create a business-oriented dashboard using Power BI

##  Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI

##  Exploratory Data Analysis
The EDA includes:
- Data cleaning and preprocessing
- Univariate and multivariate analysis
- Visual exploration of key variables

Notebook available in: `notebooks/employees_eda.ipynb`

##  Power BI Dashboard
The dashboard follows a **One Big Table** model and presents:
- Headcount by department
- Salary distribution
- Key KPIs for HR analysis
